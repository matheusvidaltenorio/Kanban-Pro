# 🧩 Kanban Pro

Aplicação de gerenciamento de tarefas estilo Trello construída com React.

## 🚀 Tecnologias

- React (Vite)
- Zustand (estado global)
- dnd-kit (drag and drop)
- Framer Motion (animações)
- LocalStorage (persistência)

## ✨ Funcionalidades

- Criar tarefas
- Deletar tarefas
- Drag & Drop entre colunas
- Reordenar tarefas dentro da mesma coluna
- Persistência automática no navegador
- Layout fixo e organizado

## 🧠 Decisões Técnicas

- Utilizei Zustand para gerenciamento global de estado pela simplicidade e performance.
- Usei dnd-kit para controle fino do drag and drop.
- Implementei persistência manual no localStorage para manter o estado entre sessões.
- Separei responsabilidades em componentes (Board, Column, Card) para manter arquitetura limpa.
